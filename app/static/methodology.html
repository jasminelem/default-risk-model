Methodology

How the default probability model works.

Data Sources

All data is sourced from WRDS. Financial statements from Compustat, stock prices from CRSP, bankruptcy filings from FJC (1929-2019) and CapIQ Key Developments (2020+), macro rates from Fed H.15.

60+ financial ratios are engineered from raw data: leverage, profitability, liquidity, Altman Z components, cash flow, market signals, and credit ratings. Companies are linked via GVKEY-PERMNO with strict date filtering to prevent data leakage.

Model

A single XGBoost classifier predicts default probability at both the 12-month and 5-year horizons. The prediction horizon is encoded as a feature (horizon_months = 12 or 60), so each company is scored twice from one model.

  • Monotonicity — XGBoost monotone_constraints guarantee P(5y) ≥ P(12m) by construction.
  • Calibration — 5-fold isotonic regression maps raw scores to true probabilities using ~400K cross-validated samples.
  • Tuning — Optuna HPO maximises ROC-AUC on a time-based validation set.

Explainability

Every prediction includes SHAP values showing which features push default probability up or down for that company. Positive value = increases risk, negative = decreases it. One explainer handles both horizons.

CDS Fair Value

Model PD is converted to an implied CDS spread using the reduced-form pricing framework:

spread = -ln(1 - PD) × (1 - R) × 10,000
PD = 1 - exp(-spread / (1 - R) / 10,000)

R = 40% recovery (senior unsecured). Assumes constant hazard rate. Indicative only.

Peer Comparison

Each company's PD is ranked against its SIC industry sector, showing the percentile rank, sector median, and five closest peers by default probability. Financials and utilities are excluded.

Calibration

Predicted PDs are binned and compared against actual default rates. Ratio near 1.0 = well calibrated.
